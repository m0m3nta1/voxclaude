# VoxClaude

Голосовой ИИ-помощник для Discord. Скажи в войсе «Клод, изменили в последнем патче Dota 2?» — и он ответит голосом.  


## ⚠️ Дисклеймер

Бот **обрабатывает весь звук в голосовом канале** — это технически необходимо для работы wake-word. Он предназначен для использования **только в каналах, где** **все участники знают о его присутствии и согласны с ним**. Команда `/join` намеренно публикует уведомление об этом в чат - не удаляйте это поведение.

Бот не сохраняет аудио: звук обрабатывается в памяти и удаляется после распознавания. Текст распознанных фраз с обращением «Клод» отправляется в Gemini API (Google) — учитывайте это в приватных разговорах.

Полностью бесплатный стек:

| Этап | Технология |
| - | - |
| Приём голоса из войса | discord-ext-voice-recv |
| Распознавание речи | faster-whisper (локально, GPU/CPU) |
| LLM | Gemini API (free tier), с Google Search grounding |
| Синтез речи | edge-tts |


## Возможности

- Wake-word «Клод» в любом месте фразы, распознавание нескольких участников войса параллельно

- `/join`, `/leave` — управление присутствием в войсе

- `/ask` — вопрос текстом (озвучит, если в войсе)

- `/learn` — база знаний, пополняемая по ходу игры и попадающая в промпт

- Автопоиск в Google при словах «патч», «нерф», «актуальный» и т.п.; принудительный — «Клод, загугли …»

- Автовыход при пустом войсе, защита от битых opus-пакетов, ретраи при перегрузе Gemini

## Установка

Требуются Python 3.11+, [ffmpeg](https://www.gyan.dev/ffmpeg/builds/) и токены: Discord bot token, [Gemini API key](https://aistudio.google.com/) (проект без биллинга — иначе free tier отключается).

```
python -m venv .venv  
.venv\\Scripts\\activate          \# Windows  
pip install -r requirements.txt  
copy .env.example .env          \# заполнить токены и путь к ffmpeg  
python main.py
```

GPU-ускорение Whisper (NVIDIA):

```
pip install nvidia-cublas-cu12 nvidia-cudnn-cu12
```

DLL из этих пакетов подхватываются автоматически при старте.

## Важное ограничение: Stage-каналы

С марта 2026 Discord требует E2EE (DAVE protocol) в обычных голосовых каналах, и приём звука в них пока не поддерживается библиотекой voice-recv ([issue](https://github.com/imayhaveborkedit/discord-ext-voice-recv/issues)). **Бот слушает только Stage-каналы (трибуны)** — они исключены из принудительного E2EE. Говорящие должны быть спикерами. Когда в voice-recv появится поддержка DAVE, обычные войсы заработают обновлением зависимости.

## Прочие известные грабли

- Free tier Gemini сильно урезан (порядка 20-250 запросов в день в зависимости от модели) - реальные квоты проекта смотрите в AI Studio

- Из некоторых стран generativelanguage.googleapis.com и speech.platform.bing.com могут быть недоступны напрямую

- Free tier Gemini может использовать запросы для обучения моделей Google

- Бот обрабатывает весь звук в канале - предупреждение встроено в /join  
  
Можно дополнить локальными моделями(Ollama и пр.)
